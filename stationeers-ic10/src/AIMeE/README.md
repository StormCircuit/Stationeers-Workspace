Instructions for using the AIMEE scripts in this folder:

1. Setup 1 roboroutine per aimee and 1 transmitter per aimee
1. Setup 1 switch of some kind per aimee for recalling them (set them to on after placing them)
3. Choose either robomaps or robomapsParallel for an army of AIMEES
3a. Note that robomapsParallel supports up to 6 aimees on a private subnet (utilizes name hashes)
4. Set the device pins as labeled
5. If using robomapsParallel, set the names of the housings containing roboroutine as required
6. Turn lever/switch off to enable aimee units
